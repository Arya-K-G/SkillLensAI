# 🚀 SkillLensAI

**SkillLensAI** is an AI-powered resume analysis and skill gap detection platform that helps users evaluate their resumes against specific job roles and improve their chances of getting hired.

---

## 📌 Overview

In today’s competitive job market, many candidates struggle to understand why their resumes don’t match job requirements. SkillLensAI solves this problem by:

* Analyzing resumes using AI
* Matching them with job roles
* Identifying missing skills
* Recommending learning resources

---

## 🎯 Key Features

* 📄 **Resume Upload & Parsing**
  Upload resumes (PDF/DOCX) and extract key information automatically.

* 🤖 **AI-Based Resume Analysis**
  Analyze resumes using AI to evaluate skill relevance.

* 🎯 **Job Role Matching**
  Compare user skills with required skills for specific job roles.

* ⚠️ **Skill Gap Detection**
  Identify missing skills required for the selected role.

* 📚 **Learning Recommendations**
  Suggest resources to improve missing skills.

* 📊 **Match Percentage**
  Show how well a resume matches a job role.

* 🕘 **Analysis History**
  Save and view previous resume analysis results.

---

## 🛠️ Tech Stack

**Frontend:**

* Flutter (Dart)

**Backend:**

* Python (Flask)

**Other Technologies:**

* REST APIs
* AI-based analysis logic
* PDF/DOCX parsing

---

## 🧠 How It Works

1. User uploads a resume
2. System extracts skills and information
3. User selects a job role
4. AI compares resume with job requirements
5. Displays:

   * Matching skills
   * Missing skills
   * Match percentage
6. Recommends learning resources

## 🚀 Installation & Setup

### Prerequisites

* Flutter SDK
* Python installed

### Steps

```bash
git clone https://github.com/Arya-K-G/SkillLensAI.git
cd SkillLensAI
```

### Run Flutter App

```bash
flutter pub get
flutter run
```

### Run Backend

```bash
cd backend
pip install -r requirements.txt
python app.py
```
